# Unbiased AI Decision

A React-based web application to detect, analyze, and mitigate bias in automated AI decisions.

## ✨ Features
- Bias Detection with multiple fairness metrics
- Interactive What-If Fairness Explorer
- Mitigation Lab with Before/After comparison
- Datasets upload and column configuration
- Audit Reports & Export
- Dark mode support
